📌 Overview
This project is a simple Job Portal Web Application built using HTML, CSS, and JavaScript.
It simulates a client-server model where the frontend interacts with mock data as if it were coming from a backend server.
The application allows users to:
  View job listings
  Search/filter jobs
  Apply for a job using a form

🚀 Features
✅ Job Listings Display
Shows available jobs in card layout
✅ Search Functionality
Filter jobs dynamically based on user input
✅ Apply Form
Users can apply using Name + Job ID
✅ Form Validation
Ensures all fields are filled correctly
✅ Dynamic UI Updates
Uses DOM manipulation for real-time updates

🛠️ Technologies Used
  HTML – Structure
  CSS – Styling & layout
  JavaScript – Logic & interactivity

🧩 Project Type
  This is a Frontend Simulation of Client-Server Architecture:
  Job data is stored locally (simulating server response)
  Form submission mimics an API request
  No actual backend/database is used

🎯 Learning Outcomes
  Understanding of DOM Manipulation
  Basics of Client-Server Model
  Form handling and validation
  Dynamic UI updates using JavaScript
